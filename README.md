# Voice-Commerce-Application

A simple yet powerful app that helps you find the perfect pair of shoes using your voice. Instead of just clicking buttons, you can talk to the app to search for styles, colors, or brands.

# üìù Project Description
This is a modern web application built with React and TypeScript. It uses Artificial Intelligence (AI) to understand what you're looking for. It connects to smart services like Gemini and Groq to make shopping feel more like a conversation than a search.

# ‚ú® Key Features
1.**Voice Search:** Use your microphone to find shoes instead of typing.

2.**Multilingual Support:** Talk or search in your preferred language‚Äîthe AI understands and responds across different languages.

3.**Lightning-Fast Voice Processing:** Uses Groq STT (Whisper Large v3) to convert speech to text in milliseconds.

4.**Intelligent Attribute Extraction:** Passes the transcribed text to Gemini LLM, which automatically identifies:

**Style:** Sneakers, Boots, Running shoes.

**Details:** Size, Color, Brand, and Gender.

5.**Detailed Views:** Get a close-up look at every shoe with specific product details.
